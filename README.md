Amazon Review Sentiment Analysis Web App

This project predicts whether Amazon musical instrument reviews are Positive, Neutral, or Negative using Machine Learning and NLP.
Built using Python, Scikit-learn, NLTK, and Streamlit with an interactive web dashboard.
Includes multiple model comparison (Logistic Regression, Naive Bayes, SVM) and confidence score prediction.
Displays dataset insights, sentiment distribution, confusion matrix, and word cloud visualization.
Run the app using: streamlit run streamlit_app.py
